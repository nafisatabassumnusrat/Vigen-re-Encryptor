# 🔐 CipherForge — Vigenère Cryptography Studio

✨ **Encrypt • Decrypt • Visualize — All in One Place**

CipherForge is a modern, interactive cryptography tool built for students, developers, and security enthusiasts. It allows real-time encryption, decryption, and deep analysis using classical ciphers like **Vigenère** and **Caesar** — all inside your browser.

---

## 🚀 Live Experience

> ⚡ No installation needed — runs directly in your browser

---

## 🌟 Features

🔐 **Dual Cipher Support**
- Vigenère Cipher (Advanced classical encryption)
- Caesar Cipher (Shift-based encryption)

⚡ **Real-Time Processing**
- Instant encryption & decryption
- Lightning-fast performance

📊 **Analytics & Visualization**
- Character mapping visualization
- Frequency analysis charts
- Message & key statistics

🧠 **Smart Key System**
- Auto key generation
- Strength indicator
- Key validation (A–Z only)

📁 **File Support**
- Upload `.txt` files
- Drag & drop support

🕘 **History Tracking**
- Stores last 5 encryption sessions
- Quick reuse functionality

🎨 **Premium UI/UX**
- Dark/Light mode toggle 🌙
- Smooth animations & ripple effects
- Glassmorphism + modern design

🔒 **Privacy First**
- 100% client-side encryption
- No server, no data tracking

## 🧩 How It Works

1. Enter your message ✍️  
2. Provide a key (or auto-generate) 🔑  
3. Choose cipher mode 🔄  
4. Click Encrypt / Decrypt ⚡  
5. View analytics & results 📊
